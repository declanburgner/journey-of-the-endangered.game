# Sprint 1 Run Guide

## 1. Install Node.js
- Install Node.js LTS from nodejs.org.

## 2. Install packages
- In this folder, run:
  - npm install

## 3. Create env file
- Copy .env.example to .env
- You can keep defaults for first test.

## 4. Start server
- Run:
  - npm run dev

## 5. Test in browser
- Open:
  - http://localhost:3000

## 6. Test login with default users
- Master passcode: family-server-123
- User 1: alice / alice-pass
- User 2: bob / bob-pass

## 7. Success checks
- You can log in as both users.
- You are redirected to main page.
- Main page shows a welcome message from /api/main.
